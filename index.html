About The Eye Project*

Hello! I am Bryan Schrock, and I created this. If you're interested, you can view the source codes here.

You can go back to the project at any time by pressing M. Also, you can press it again to come back here.

How it works

The Eye Project started as a sort of experiment in randomness. Originally, it generated a single Eye in the center of the canvas, with random values governing most of the parameters for creation. Later versions became more interactive, and finally I added the automatic tutorial to tell you how to use it.

The project runs best on modern Google Chrome browsers. The Color Painter functionality is an HTML5 device, and I didn't try very hard to make IE or Firefox supported. Sorry. Use Chrome.

So, read the text in the automatic tuorials. It's very helpful I think, if you know the terms:

  1. Eye: The circular shape that is created upon clicking. An Eye has a single pupil, and many rings.

  2. Pupil: The center of the Eye, like the hole in a donut. There are many functions that can only be performed by clicking and holding the pupil. You can:
    • Grow the entire Eye itself, linearly scaling the entire thing up or down. Do it with Q and W.
    • Scale the size of the pupil only. Do it with A and S.
    • Increasing or decreasing the number of vertices in the very first ring. If you click on the ring and increase the number of vertices, you actually only grow the second part of the ring's number of vertices- so this function has to go here. Do it with Z and X.
    • Add a new ring to the outside of the Eye. It will be White and will not be tagged for the Color Painter- more on that later. Do it with E.
    • Delete the last ring from the Eye. Do it with D.
    • Reset all of the Color Painter's points. More on that later. Do it with R.

  3. Ring: Each section of triangles that surround each other and the pupil is a ring. The number of rings is random but can be changed manually. See above for that. Each Ring has an evenly spaced number of points on it's outside edge, and the triangles are drawn between the points around the outside edge, and the two closest points in the next previous Ring's set of points. There are many functions that can only be accomplished by clicking and holding a Ring:
    • Scale the Ring's width. Do it by moving your mouse left or right.
    • Delete the Ring. Do it with D. The next biggest Ring will move in to take its place.
    • Change the number of vertices in the Ring. The number of points that are along the Ring's outside edge can be modified with W and Q while holding the Ring. Doing this will change the appearance of the next Ring as well.
    • Apply color. With A, you can apply the Color Painter's color.
    • Sample color. With S, you can set the Color Painter's color to be that of the selected Ring.

  4. Color Painter: The Color Painter is a device for modifying the colors in your Eye. Painting a Ring will "mark" the Ring for the Color Painter. The Color Painter then goes through the Eye and calculates a gradient for every Ring that is not marked, based on the two closest marked ring's colors. The Color Painter is not supported by Internet Explorer, Safari, or Firefox. again, try Chrome
  5. Thumbnail Manager: The order that your eyes lie can be modified using the Thumbnail Manager. Press T to toggle it. Click and drag a thumbnail to change where it is in relation to the other Eyes. The lower in the manager, the closer to the front the Eye will be.
